###Installing Dependencies and Running Project
1. Need React-Native CLI.
2. Need NPM.
3. Need Node version>8
4. After that just hit 'npm install' and then 'react-native run-android' for android and 'react-native run-ios' for IOS.

**Another thing could cause problem that is SDK location. Be sure to change it in the 'android/local.properties' .

you are good to go.
